My Hobbies

Below is a list of things I like to do on my spare time. They are not listed in any specific order.

  • Arts and Crafts

    • I really enjoy spending time creating things. My main go to is knitting and cross-stich. Recently I started learning how to tat(not the same as tattooing). I learned about a new craft called "Swedish Weaving" that I really want to tryout.

  • Web Development

    • I started out creating my own websites using XAMPP/WordPress just for the fun of trying. I never expected it to really become more then just something to learn. It became a way for me to cope with depression and even saved my life.

  • Programming

    • Programming became another coping tool I picked up. I first began with C++ and moved onto Java, and then finally to Python. I find Python to be my favorite as it is the easiest for me to understand and I have managed to create more personal software programs with it then other languages I have learned.
